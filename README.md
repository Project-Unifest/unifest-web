## 시작하기

### 개발 서버 실생하기

의존성을 설치하고 개발 서버를 실행합니다.

```bash
yarn install
yarn dev
```

### 테스트 실행하기

#### Storybook 테스트 실행하기

Component Testing 및 Interaction Testing을 진행합니다.

```base
yarn storybook
yarn test-storybook
```

#### E2E 테스트 실행하기 

Playwright을 이용하여 E2E 테스팅을 진행합니다.

```bash
yarn e2e
```

### 빌드하기

다음 명령어를 실행하세요.

```bash
yarn build
```
